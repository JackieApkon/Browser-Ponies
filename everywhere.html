      About

      This is a version of Browser Ponies that lets you put ponies on a website by simply putting "https://tinyurl.com/ponyfy2#" in front of the URL of the website. So e.g. "example.com" becomes "https://tinyurl.com/ponyfy2#example.com".

      Note that not all websites allow embedding in an iframe (which is used here). E.g. Google does not work.

      See this for more information and a way to embed ponies to your website permanently.
